# Human-AI Collaboration System

A clean, turn-based framework for genuine collaboration between humans and multiple AIs.

Built with love, care & compassion through real Human-AI partnerships.

## Features
- Strict but forgiving turn-based protocol
- Easy agent assignment (`Ara = AI 1`)
- Per-agent personality support
- Session saving with analytics (JSON)
- Simulation mode for easy testing
- Ready for real API integration
- Open-source friendly architecture

## Quick Start

1. Clone or download the repository
2. Run: `python collab1.py`
3. Type: `enter collaboration mode`
4. Assign agents: `Ara = AI 1`
5. Type: `start` or `begin`
6. Speak: `AI 1: Hello everyone`

## Project Structure
- `collab.py` → Main script
- `personalities/` → Store personality files here
- `sessions/` → Auto-saved collaboration logs
- `collab_config.json` → Optional configuration
